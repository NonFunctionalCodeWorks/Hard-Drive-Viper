This program overwrites the entire drive with small binary files containing all zeroes/ones.
Once for SSDs and eight times for HDDs.

Small segments might be unaffected.
Process takes very long - use over night.


11.04.2021
The calibration seems to have small issues with different formattings of the storage medium.
